# Imágenes para Open Graph

## default-og.jpg
Para mejorar el SEO del blog, necesitas agregar una imagen por defecto para Open Graph.

**Especificaciones recomendadas:**
- Nombre: `default-og.jpg`
- Tamaño: 1200x630 píxeles (ratio 1.91:1)
- Formato: JPG o PNG
- Peso: Menos de 1MB
- Contenido: Logo de NachoWeb3 + texto descriptivo del blog

**Ejemplo de contenido:**
- Logo de NachoWeb3
- Texto: "Noticias de IA, Blockchain y las mejores guías"
- Colores de marca consistentes con el diseño del blog

## logo.png
También se recomienda agregar un logo para Schema.org structured data.

**Especificaciones:**
- Nombre: `logo.png`
- Tamaño: 600x600 píxeles (cuadrado)
- Formato: PNG con transparencia
- Peso: Menos de 200KB
